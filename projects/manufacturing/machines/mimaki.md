# Mimaki — UJF-6042 MkII A2 UV-LED flatbed (EOL)

The Mimaki is NBNE's smaller A2 UV-LED flatbed printer, made in
Japan by Mimaki Engineering, distributed in the UK by Hybrid
Services (Crewe). Direct-to-object printing on rigid substrates
up to 153 mm thick, 610 × 420 mm print area, on-demand piezo
head with circulation technology + nozzle recovery system + MAPS
anti-banding. CMYK + White + Clear varnish + Primer ink
configuration (specific config TBD).

**End-of-life status declared.** Currently used only for black-
and-white production while the larger Refine Color UV machines
handle full-colour work. A regulatory ink discontinuation hit on
2026-03-31 (Mimaki ended sales of LH-100 and LF-140 inks under
EU REACH/SVHC compliance pressure), which forces a near-term
decision: convert to ELH-100 (CMR-free replacement, free
conversion programme from Mimaki, ~8 hours largely unattended,
keeps the machine running indefinitely), run down stockpiled
LH-100 (no further investment but no path forward beyond
exhaustion), or accelerate retirement (clean ending, exits the
EOL niche, possible Hybrid trade-in support).

The decision framework, the three options, the input data
needed, and the dated audit trail of how the decision develops
all live in `mimaki-eol-decision.md` — append-only as Hybrid
quotes come in and stock burn-rate is measured.

## Tribal knowledge — different shape vs other machines

For long-horizon machines (Hulk, Roland, Mutoh, Epson), the
tribal-knowledge harvest is about capturing what operators have
learned for ongoing use. For the Mimaki, the harvest is about
**capturing what the machine has TAUGHT NBNE before it retires**
— substrate-behaviour knowledge that transfers to other UV
printers even after the Mimaki goes. Similarly the RasterLink
profile library is institutional knowledge that may inform Rolf
/ Mao profile work on the same substrates.

Same 30-min recorded session pattern, but framed differently:
"What do you know about the Mimaki that we shouldn't lose when
it goes?"
