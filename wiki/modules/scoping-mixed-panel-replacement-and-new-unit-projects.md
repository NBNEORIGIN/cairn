# Scoping Mixed Panel Replacement and New Unit Projects

This article covers how to correctly scope quotations for projects that involve both new display units (complete with posts/frames) and panel-only replacements into existing structures. Getting this distinction right at the enquiry stage ensures accurate pricing, prevents scope creep, and avoids installation complications. The Berwick-upon-Tweed tourist maps case study demonstrates the critical importance of clarifying this early.

## Initial Enquiry Assessment

When a customer requests pricing for multiple display locations, immediately assess whether the project involves:

1. **New complete units** - Frame, posts, foundations, and panels
2. **Panel replacements** - New graphics into existing display structures
3. **Mixed scope** - Combination of both

**⚠️ WARNING**: Never assume all locations are the same type. Customers often don't distinguish between these in initial specifications, leading to significant pricing discrepancies.

## Clarification Process

### Step 1: Request Explicit Breakdown

Contact the customer directly and ask them to identify which category each location falls into. Use clear, simple language:

- "Which locations require **full new units** (frame + posts)?"
- "Which are **panel replacements** into existing structures?"

### Step 2: Document Customer Response

Record the exact breakdown provided. In the Berwick example:

- **Location C (Parade Car Park)**: Panel replacement - attaching to existing structure
- **Other locations**: Status not yet confirmed - requires follow-up

### Step 3: Flag Ambiguities

If the customer only confirms part of the scope (as in the example), immediately request clarification on remaining locations before proceeding with quotation.

## Site-Specific Considerations

### Existing Structure Assessments

When panels are replacing into existing structures, verify:

- **Structural condition** - Is the existing frame suitable for reuse?
- **Fixing compatibility** - Will new panels fit existing mounting systems?
- **Access requirements** - Can existing structures be accessed safely?
- **Material compatibility** - Are existing frames compatible with proposed panel materials?

### New Unit Requirements

For locations requiring complete new units, confirm:

- Ground conditions for post installation
- Planning/highways permissions
- Required foundation depth
- Proximity to utilities
- Vehicle access for installation

## Pricing Structure

### Create Separate Line Items

Always structure quotations with distinct sections:

**Section A: Panel Replacements**
- Supply of panels only
- Installation to existing structures
- Unit rate per location

**Section B: New Complete Units**
- Supply of frame and posts
- Supply of panels
- Groundworks and installation
- Unit rate per location

This transparency allows customers to understand cost differences and makes variations easier to manage.

### Cost Implications

Typical cost ratios to consider:

- **Panel replacement**: 30-40% of new unit cost
- **New complete unit**: 100% (baseline)
- **Mixed projects**: Calculate weighted average based on quantity split

**⚠️ PITFALL**: Quoting an average price across all locations obscures the true scope and causes problems if the customer later changes the mix of replacements vs. new units.

## Common Customer Misunderstandings

### "It's All the Same"

Customers often view all locations as equivalent because they're receiving the same artwork. Educate them on the difference between:

- **What they see** (the graphic panel)
- **What supports it** (the infrastructure)

### Attachment vs. Installation

The phrase "attaching to an existing structure" (as in Location C) is a key indicator of panel replacement scope. However, verify this assumption - sometimes customers mean attaching a new unit *adjacent to* an existing structure.

### Artwork Provision

When customers provide artwork (as Berwick TC did), confirm:

- File formats and specifications
- Whether artwork is sized for quoted panel dimensions
- Approval process and timeline
- Who is responsible for any required modifications

## Documentation Best Practices

1. **Email confirmation** of scope breakdown before quoting
2. **Photographic evidence** where possible (request site photos)
3. **Site visit recommendations** for mixed projects over £5k
4. **Clear quotation notes** explaining assumptions made

## Follow-Up Actions

After receiving partial clarification (like the Berwick example):

1. Acknowledge the confirmed information
2. Reiterate outstanding questions
3. Provide a timeline for quotation once full scope is confirmed
4. Offer a site visit if complexity warrants it

**⚠️ WARNING**: Do not proceed with quotation on incomplete scope information. This leads to repricing, credibility issues, and potential margin erosion.
