# Re: Pricing

## Summary

Email thread with Berwick-upon-Tweed Town Council regarding a quotation for tourist direction maps. The project involves three locations with different installation requirements: one panel replacement into existing structure (Location C - Parade Car Park) and two full new installations requiring posts. Customer is providing artwork. Thread demonstrates the critical need to clarify installation scope before pricing to avoid quotation errors.

## Key Project Details

**Customer:** Berwick-upon-Tweed Town Council  
**Contact:** Stephen Robinson (Operations Manager)  
**Project Type:** Tourist direction maps - supply and installation  
**Locations:** 3x sites in Berwick-upon-Tweed  
**Artwork:** Customer-supplied

### Installation Breakdown

- **Location C (Parade Car Park):** Panel replacement into existing structure only
- **Locations A & B:** Full new installations (assumed to require complete units with posts)

## Process Steps

### 1. Initial Enquiry Review

When customer requests pricing for "supply and installation" of directional signage:

- Check if specifications or PO descriptions are attached
- Look for any mention of existing structures
- Note whether customer is providing artwork (affects lead time and pricing)

### 2. Scope Clarification (Critical Step)

Before pricing, always establish:

1. Which locations have existing display structures suitable for panel replacement
2. Which locations require full new units (frame + posts + groundworks)
3. Panel sizes for each location
4. Ground conditions for new posts (if applicable)

**In this case:** Sales team correctly identified potential mixed scope and requested clarification before quoting.

### 3. Customer Response Analysis

Stephen confirmed:

- Only 1 of 3 locations attaches to existing structure (Location C)
- Image provided showing installation area
- Implies remaining 2 locations need full new installations

### 4. Next Actions for Quotation

With scope now clarified, proceed to:

1. Price Location C as panel replacement only
2. Price Locations A & B as complete new installations including:
   - Display frames
   - Posts/support structures
   - Groundworks/installation
3. Separate line items on quotation for clarity
4. If image shows ground conditions, assess for access/installation considerations
5. Note artwork is customer-supplied in quotation assumptions

## Warnings and Common Pitfalls

⚠️ **Never assume installation scope** - "Supply and installation" can mean:
- New standalone units
- Panel replacements
- Additions to existing structures
- Mix of all three

⚠️ **Pricing errors to avoid:**
- Quoting full units when customer only needs panels
- Quoting panels only when groundworks required
- Not separating different scope items on quotation

⚠️ **Cost implications are significant:**
- Panel replacement: £X (materials + fitting only)
- Full new installation: £X (materials + posts + groundworks + installation)
- Difference can be several hundred pounds per location

⚠️ **Question everything in PO descriptions:**
- Council PO systems often use generic descriptions
- Don't rely on purchase order wording alone
- Always verify directly with customer contact

## Best Practices from This Thread

✓ **Proactive clarification:** Sales team identified ambiguity and asked before pricing  
✓ **Specific questions:** Asked which locations need what, not vague "please clarify"  
✓ **Business justification:** Explained why clarification needed ("ensure most cost-effective approach")  
✓ **Professional tone:** Positioned as helping customer get accurate pricing

## Customer-Supplied Artwork Considerations

When artwork is customer-supplied:

- Confirm file format requirements upfront (.pdf, .ai, etc.)
- Specify resolution/quality requirements
- Request artwork well in advance of production date
- Note on quotation: "Subject to artwork approval for print suitability"
- Build in revision allowance if artwork needs adjustment
