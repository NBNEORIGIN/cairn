# Kit's GCSE Project

This article documents the setup and processing of Kit's GCSE project, processed as a customer job involving DXF files for cutting on clear acrylic. The project serves as a reference example for handling educational customer projects with externally-provided design files on portable media.

## Project Overview

**Customer:** Vicky (parent/guardian)  
**Material:** 3mm or 4mm clear acrylic  
**File Format:** .dxf  
**Delivery Method:** Email + memory stick backup  
**Appointment Time:** 10:30

## Pre-Appointment Preparation

1. **Review emailed DXF files** before the customer arrives to identify any potential issues
2. **Check file compatibility** with Deek cutting software
3. **Prepare material samples** of both 3mm and 4mm clear acrylic for customer inspection
4. **Calculate approximate cutting time** based on design complexity
5. **Estimate material costs** for both thickness options

## Customer Consultation Steps

1. **Verify file integrity** - Check both emailed files and memory stick versions match
2. **Discuss material selection** - Help customer choose between 3mm and 4mm based on:
   - Project structural requirements
   - Budget constraints
   - Aesthetic preferences
   - GCSE project assessment criteria (if applicable)
3. **Review design** with customer to confirm cutting expectations
4. **Provide cost estimate** including material and machine time
5. **Agree timeline** for completion (consider GCSE submission deadlines)

## File Handling

### DXF Import Procedure

1. Open files in Deek software
2. Check units are correct (mm vs inches)
3. Verify layer assignments for cutting/engraving operations
4. Confirm all paths are closed where required
5. Check for duplicate lines or overlapping vectors

### Common DXF Issues

- **Disconnected paths** - Educational design software sometimes creates gaps in what appear to be continuous lines
- **Incorrect scale** - Always verify dimensions with customer before cutting
- **Multiple layers** - Ensure cutting order is logical and all necessary layers are enabled
- **Text elements** - Confirm whether text should be cut, engraved, or outlined

## Material Preparation

### Clear Acrylic Considerations

- Remove protective masking carefully after cutting (especially important for GCSE presentation)
- **Warning:** Clear acrylic shows fingerprints and scratches easily - handle with clean cotton gloves
- Consider cutting orientation to minimize visible tooling marks on display faces
- Check material is flat and free from warping before cutting

### Thickness Selection Guidance

**3mm acrylic:**
- More economical
- Easier to cut (faster job)
- Suitable for display pieces or lighter assemblies

**4mm acrylic:**
- More robust
- Better for structural projects
- Slightly more professional appearance

## Cutting Process

1. **Load correct material** on bed
2. **Secure material** ensuring it's flat (use weights if necessary)
3. **Set appropriate cutting parameters** for acrylic thickness
4. **Perform test cut** on scrap if unfamiliar with the design
5. **Monitor first few cuts** to ensure proper depth and quality
6. **Do not leave machine unattended** when cutting acrylic

> **Warning:** Acrylic can produce fumes. Ensure extraction is operating correctly throughout the job.

## Quality Control

- Inspect all cuts for completeness
- Check corners are fully cut through
- Verify dimensions match design specifications
- Test-fit any interlocking components before customer collection
- Remove debris and clean parts carefully

## Post-Cutting

1. Leave protective masking on until customer collection
2. Package carefully to prevent scratching
3. Include any offcuts if customer requests (material evidence for GCSE portfolio)
4. Provide basic care instructions for acrylic

## Pricing Considerations

- Educational projects may qualify for reduced rates (check current NBNE policy)
- Factor in design consultation time
- Include material wastage in estimate
- Consider rush fees if GCSE deadline is imminent

## Common Pitfalls

- **Assuming file dimensions are correct** - always verify with customer
- **Not discussing deadline** - GCSE projects often have hard submission dates
- **Inadequate file backup** - memory stick files can be corrupted; verify email versions work
- **Rushing the material discussion** - choosing wrong thickness affects project success
- **Not keeping offcuts** - students sometimes need material samples for documentation
