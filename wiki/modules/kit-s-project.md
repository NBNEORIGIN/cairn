# Kit's Project

## Summary

Kit's Project refers to a GCSE educational project requiring CNC machining of clear acrylic components. The project files were provided as .dxf format, with material specifications of 3-4mm clear acrylic. This serves as a reference case for handling external educational projects brought to NBNE, particularly those involving student work with specific academic deadlines and material requirements.

## Project Specifications

**Material**: 3mm or 4mm clear acrylic (cast acrylic recommended)  
**File Format**: .dxf (AutoCAD Drawing Exchange Format)  
**File Delivery**: Email attachment plus physical backup on memory stick  
**Client Contact**: Vicky (parent/guardian)  
**Timeline**: Discussed in person at 10:30 appointment

## Workflow Steps

### 1. File Reception and Backup

- Receive .dxf files via email attachment
- Request physical backup on USB/memory stick as redundancy
- Save files to project folder immediately upon receipt
- Verify files open correctly in CAM software before client meeting

### 2. Pre-Meeting Preparation

- Open .dxf files in appropriate CAM software (VCarve Pro or similar)
- Check for any file corruption or missing elements
- Assess cutting requirements (through-cuts, engraving, etc.)
- Prepare material samples of 3mm and 4mm clear acrylic for client to evaluate
- Calculate preliminary time and cost estimates

### 3. Client Consultation

- Discuss material thickness choice (3mm vs 4mm) based on:
  - Structural requirements of the design
  - Aesthetic preferences
  - Budget constraints
  - Weight considerations
- Confirm final dimensions and any design modifications
- Establish completion deadline (critical for GCSE projects)
- Agree on pricing and payment terms
- Obtain approval to proceed

### 4. CAM Programming

- Import .dxf files into CAM software
- Set appropriate toolpaths for clear acrylic:
  - Use sharp, new bits to minimize edge crazing
  - Single-flute or O-flute cutters recommended for acrylic
  - Set conservative feed rates to prevent melting
- Add tabs if necessary to prevent part movement during cutting
- Generate and verify G-code

### 5. Material Preparation

- Use **cast acrylic** rather than extruded for better edge finish
- Check material is free from scratches on protective film
- Ensure material is flat and properly secured to bed
- Verify material thickness matches CAM settings

### 6. Machining

- Run test cut on scrap acrylic if possible
- Monitor first few passes to ensure proper cutting
- Watch for chip evacuation (chips should be long spirals, not dust)
- Pause and clear chips if necessary to prevent re-melting

### 7. Post-Processing

- Carefully remove parts from bed
- Remove protective film
- Flame-polish edges if required (discuss with client first)
- Inspect for any defects or incomplete cuts
- Package carefully to prevent scratching

## Warnings and Common Pitfalls

**⚠️ Educational Project Deadlines**: GCSE projects have strict submission deadlines that cannot be extended. Confirm the deadline explicitly and build in buffer time for any issues.

**⚠️ Acrylic Melting**: Acrylic melts easily if cutting parameters are incorrect. Blunt tools, excessive speed, or poor chip evacuation will cause melted edges requiring extensive rework.

**⚠️ File Format Issues**: .dxf files may contain layers, duplicated lines, or scaling issues. Always verify dimensions and clean up drawings before generating toolpaths.

**⚠️ Material Scratching**: Clear acrylic shows every scratch. Keep protective film on as long as possible and handle with clean gloves during final inspection.

**⚠️ Backup Files**: Always insist on physical backup media for important projects. Email attachments can be corrupted or lost.

## Material Selection Guidance

**3mm Clear Acrylic**:
- More flexible
- Lighter weight
- Easier to cut cleanly
- May be too thin for structural elements

**4mm Clear Acrylic**:
- Better structural rigidity
- More professional appearance
- Slightly harder to cut without edge crazing
- Higher material cost
