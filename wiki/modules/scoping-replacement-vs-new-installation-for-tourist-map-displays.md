# Scoping Replacement vs New Installation for Tourist Map Displays

This article covers how to correctly scope quotations for tourist information boards and wayfinding signage when the enquiry involves mixed installation types. Based on correspondence with Berwick-upon-Tweed Town Council (April 2026), it demonstrates the critical importance of clarifying whether locations require full new units (posts + frames) or panel replacements into existing structures before pricing.

## Project Background

**Customer:** Berwick-upon-Tweed Town Council  
**Contact:** Stephen Robinson, Operations Manager  
**Scope:** Supply and installation of tourist direction maps across multiple locations  
**Key Detail:** Artwork to be supplied by client

## Initial Enquiry Assessment

When the initial enquiry arrived, the purchase order descriptions suggested a mixed scope that could include:

- Replacement panels within existing display structures
- Complete new display units with posts and foundations

**⚠️ WARNING:** Never assume the installation type from PO descriptions alone. The cost difference between panel replacement and full new installation can be substantial (foundations, posts, structural work vs simple panel swap).

## Clarification Process

### Step 1: Identify Ambiguity in Scope

Review the customer's specifications for any indication of:
- "Replacement" language
- References to existing structures
- Site location descriptions that might indicate new vs existing infrastructure

### Step 2: Send Structured Clarification Request

Ask specifically:
1. Which locations require **full new units** (frame + posts)?
2. Which are **panel replacements** into existing structures?

Include rationale: "This will allow me to structure the quotation correctly and ensure the most cost-effective approach."

### Step 3: Document Customer Response

In this case, the customer confirmed:
- **Only C - Parade Car Park** is attaching to an existing structure
- By implication, all other locations require full new installations
- Customer provided site imagery for verification

## Quotation Structuring

Once clarification is received, structure the quotation as follows:

### New Unit Installations (Locations A, B, D, etc.)
Price to include:
- Posts and structural framework
- Display case/frame
- Panel and graphics
- Foundation works
- Full installation labour

### Panel Replacement (Location C - Parade Car Park)
Price to include:
- Panel and graphics only
- Removal of existing panel
- Installation into existing structure
- Any refurbishment of existing frame if required

## Key Takeaways

1. **Always clarify installation type** before pricing mixed-scope signage projects
2. **Document the response** - create a clear audit trail showing which locations are which type
3. **Request site images** where possible to verify existing structure condition
4. **Structure quotations clearly** with separate line items for different installation types
5. **Check if artwork is client-supplied** - this affects your scope and pricing

## Common Pitfalls

❌ **Assuming all locations are the same type** - can lead to significant pricing errors  
❌ **Quoting full installations when replacements are needed** - makes you uncompetitive  
❌ **Quoting replacements when new installations are needed** - leads to project losses  
❌ **Not requesting site imagery** - may miss critical structural or access issues  
❌ **Unclear quotation line items** - causes confusion during project delivery

## Cost Implications

The difference in pricing between these installation types is significant:

- **Full new unit:** Includes groundworks, posts, concrete, structural frame, labour for full installation
- **Panel replacement:** Graphics production and swap-out labour only (typically 70-85% less than new installation)
