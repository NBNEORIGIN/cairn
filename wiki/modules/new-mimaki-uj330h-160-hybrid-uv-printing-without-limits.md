## Workflow Integration

### Setting Up for Rigid Media
1. Ensure the print bed is clean and level
2. Load rigid substrate onto the flatbed table
3. Adjust print height settings based on substrate thickness
4. Run test print to verify registration and ink adhesion
5. Begin production run

### Setting Up for Roll Media
1. Remove or retract rigid media handling components
2. Load roll media onto the feed system
3. Thread material through the printer path
4. Calibrate tension and take-up settings
5. Verify alignment before production

### UV Ink Considerations
- **Curing**: UV ink cures instantly upon exposure to UV light
- **Adhesion**: Test ink adhesion on unfamiliar substrates before production runs
- **Maintenance**: UV print heads require regular cleaning to prevent clogging
- **Storage**: Keep UV inks away from direct sunlight when not in use

## Warnings and Common Pitfalls

⚠️ **CRITICAL SAFETY WARNINGS**
- **UV exposure**: Never look directly at UV lamps during operation
- **Ventilation**: Ensure adequate ventilation when printing, especially on certain plastics
- **Material compatibility**: Not all materials are suitable for UV printing—always test first

### Common Operational Issues

**Problem: Poor ink adhesion**
- Cause: Inadequate surface preparation or incompatible substrate
- Solution: Clean substrate, verify material compatibility, adjust UV lamp intensity

**Problem: Registration drift between passes**
- Cause: Temperature fluctuations or media movement
- Solution: Allow machine to reach operating temperature, verify substrate is secured

**Problem: Ink buildup on print heads**
- Cause: Insufficient maintenance or incorrect environmental conditions
- Solution: Perform daily head cleaning routines, maintain recommended temperature/humidity

**Problem: Slow substrate changeover**
- Cause: Inexperienced operators or incomplete preparation
- Solution: Follow documented changeover procedures, prepare next substrate while current job finishes

## Maintenance Requirements

### Daily

- Replace consumables as needed
- Review error logs for recurring issues
- Update firmware if available
