# Pricing

## Summary

This case demonstrates the importance of clarifying the scope between **panel replacements** and **full new installations** before quoting tourist direction maps. The client (Berwick-upon-Tweed Town Council) requested pricing for multiple tourist information displays, but the initial specifications were ambiguous about whether existing structures would be reused. Toby correctly identified the scope risk and requested clarification, learning that only one location (C - Parade Car Park) would attach to an existing structure, while others presumably required full new units.

## Key Steps in Scope Clarification

### 1. Review the Initial Request
- Client requests pricing for "supply and installation" of tourist direction maps
- Artwork to be client-supplied
- Specifications include sizes and location details
- **Warning:** Generic phrases like "supply and installation" can mean different things depending on existing infrastructure

### 2. Identify Ambiguity Indicators
Look for these red flags in specifications:
- Multiple locations listed without installation method specified
- Mix of different sizes that might suggest different installation types
- No mention of whether structures exist on-site
- Vague language around "display" vs "unit" vs "panel"

### 3. Ask Targeted Clarification Questions
Toby's approach demonstrates best practice:
- Acknowledge receipt of specifications professionally
- Explain *why* clarification is needed (accurate pricing)
- Ask specific binary questions:
  - Which locations need full new units (frame + posts)?
  - Which are panel replacements into existing structures?
- Frame it as providing "the most cost-effective approach" for the client

### 4. Document the Client's Response
Stephen's response clarified:
- **Only Location C (Parade Car Park)** attaches to existing structure
- By implication, other locations require full new installations
- Client provided visual reference (image attachment)

## Pricing Implications

### Cost Difference Between Installation Types

**Full New Unit:**
- Aluminium posts/framework
- Groundworks (concrete foundations or surface mounting)
- Hardware and fixings
- Labour for installation (digging, setting, alignment)
- Typically 2-3x more expensive than panel replacement

**Panel Replacement:**
- New printed panel only
- Minimal fixings/hardware
- Quick installation into existing frame
- Assumes existing structure is sound and compliant

### What to Include in Quote Structure

Once scope is confirmed:
1. **Separate line items** for each location type
2. Clear labeling (e.g., "Location C - Panel Replacement" vs "Locations A, B, D - Full New Units")
3. Different installation rates for each type
4. Site survey provision if existing structure condition is unknown
5. Any exclusions (e.g., repairs to existing structures, utility surveys)

## Common Pitfalls

⚠️ **Assuming all locations are the same type** - Always clarify explicitly rather than inferring from limited information.

⚠️ **Quoting too early** - Rushing to provide a price without scope clarity leads to disputes, variations, or profit loss.

⚠️ **Not requesting site photos** - Stephen proactively sent an image for Location C. If clients don't offer photos, request them for any "existing structure" installations.

⚠️ **Underestimating existing structure assessment** - Even panel replacements may require structural inspection if displays are old or weather-damaged.

⚠️ **Not clarifying "supply and installation of artwork"** - Confirm whether client provides print-ready files or if design services are needed.

## Best Practice Checklist

Before pricing multi-location signage projects:

- [ ] Confirm installation method for each location
- [ ] Request site photos where existing structures mentioned
- [ ] Verify artwork specification (file format, resolution, print-ready status)
- [ ] Identify if planning permission or landlord consent required
- [ ] Check access requirements (parking restrictions, pedestrian areas)
- [ ] Confirm if utility surveys needed for groundworks
- [ ] Establish project timeline and any phasing requirements
